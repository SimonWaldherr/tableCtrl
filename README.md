#tableCtrl

make, modify, manage, export and delete HTML tables

##functions

* add (sub-)table
* add row
* add column
* change content of cell
* merge cells
* export to csv
* export to sql
* export to md
* delete row
* delete column
* delete table
* csv2table

Demo: [here](http://simon.waldherr.eu/projects/tablectrl/)
